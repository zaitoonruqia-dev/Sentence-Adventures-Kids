# Sentence Adventure Kids

Sentence Adventure Kids is an interactive Android application designed to help children learn sentence structure through AI-powered content and engaging mini-games.

## Features
- **AI-Powered Learning**: Dynamic sentence generation using Groq API (Llama 3.1) tailored to different reading levels.
- **Reading Journey**: Structured levels from toddler-friendly 2-word sentences to complete stories.
- **Interactive Games**: A "Sentence Builder" mini-game where kids can reconstruct the sentences they've learned.
- **Progress Tracking**: An interactive map showing world progression and a rewards system with stars, coins, and badges.
- **Parent Dashboard**: Detailed statistics and controls for parents to monitor their child's progress.

## Tech Stack
- **Jetpack Compose**: Modern UI toolkit for building native Android interfaces.
- **Retrofit & OkHttp**: Networking for AI API integration.
- **Groq API**: Powered by Llama 3.1 for intelligent content generation.
- **Navigation Compose**: Seamless screen transitions.
- **StateFlow & ViewModel**: Robust state management.

## Setup
1. Clone the repository.
2. Add your Groq API key in `SentenceRepository.kt`.
3. Build and run using Android Studio.
